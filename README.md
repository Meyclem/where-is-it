# ❓❗Where is it ❗❓

## Usage

### Installation

This project uses node `14.4.0` and yarn `1.22.4`.
You can manuall install them or use [asdf](https://github.com/asdf-vm/asdf).

```bash
asdf install
yarn install
```

### Start the main file

```bash
# Development mode
yarn dev

# Production mode
yarn start
```

### Build the project

```bash
yarn build
```

### Linter

You can use eslint to help you format your code.

```bash
yarn lint
```
